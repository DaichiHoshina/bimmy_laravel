# BiMMy

BiMMyは、日本各地の「おいしいもの」を投稿して共有するためのサイトです。
![alt]https://gyazo.com/b9f6b51f8410c1b4f8cf7d8de2f54451


##サイトURL
https://hidden-hamlet-51542.herokuapp.com

##主な機能

1.ユーザー登録機能
2.ユーザーログイン機能
3.ユーザープロフィール編集機能(画像付き)
4.ユーザー投稿一覧機能
4.写真&コメント投稿機能
5.いいね機能(カウント付き)

## 使用技術
